Welcome to To-Do Flask App : API

Signup User


URL : https://t0d0-flask-app.herokuapp.com/user/signup

Input

            {
                "name":"Test",
                "email":"test@demo.com"
            }
Output

            {
                "msg": "New user created : User(id=1, name=Test, email=test@demo.com)",
                "user_id": 1
            }

Delete User


URL : https://t0d0-flask-app.herokuapp.com/user/delete

Input

            {
                "user_id": 1
            }
Output

            {
                "msg": "User deleted : User(id=1, name=Sanjay, email=sanjaybabu618@gmail.com)"
            }

Add Todo


URL : https://t0d0-flask-app.herokuapp.com/todo/add

Input

            {
                "content":"Sample todo",
                "due_date":"01-05-2022 13:30:01",
                "is_completed":false,
                "user_id":1
            }
Output

            {
                "msg": "New todo added : Todo(id=1, content=Sample todo, due_date=01-05-2022 13:30:01, is_completed=False, user_id=1)",
                "todo_id": 1
            }

List Todo

            }

Get Todo


URL : https://t0d0-flask-app.herokuapp.com/todo/<todo_id>

Input

            None
Output

            {
                "msg": "Todo fetched successfully : Todo(id=1, content=Sample todo, due_date=01-05-2022 13:30:01, is_completed=False, user_id=1)",
                "todo": {
                    "content": "Sample todo",
                    "due_date": "01-05-2022 13:30:01",
                    "id": 1,
                    "is_completed": false,
                    "user_id": 1
                }
            }

Update Todo


URL : https://t0d0-flask-app.herokuapp.com/todo/<todo_id>/update

Input

            {
                "content":"This is great",
                "due_date":"09-04-2022 13:30:01",
                "is_completed":true,
                "user_id":1
            }
Output

            {
                "msg": "Todo updated successfully : Todo(id=1, content=This is great, due_date=09-04-2022 13:30:01, is_completed=True, user_id=1)",
                "todo": {
                    "content": "This is great",
                    "due_date": "09-04-2022 13:30:01",
                    "id": 1,
                    "is_completed": true,
                    "user_id": 1
                }
            }
