Welcome to To-Do Flask App : API

Signup User


URL : https://t0d0-flask-app.herokuapp.com/user/signup

Input

            {
                "name":"Sanjay",
                "email":"sanjaybabu618@gmail.com"
            }
Output

            {
                "msg": "New user created : User(id=1, name=Sanjay, email=sanjaybabu618@gmail.com)",
                "user_id": 1
            }

Delete User


URL : https://t0d0-flask-app.herokuapp.com/user/delete

Input

            {
                "user_id": 1
            }
Output

            {
                "msg": "User deleted : User(id=1, name=Sanjay, email=sanjaybabu618@gmail.com)"
            }
